# Expense Tracker (Python)

A simple command-line expense tracker built using Python to record, manage, and analyze daily expenses.

## Features
- Add expenses with amount, category, and date
- View all recorded expenses
- Calculate total spending
- Category-wise spending summary
- Delete expenses

## Technologies Used
- Python
- Lists & Dictionaries
- Functions and Loops

## How to Run
1. Download the file
2. Run using Python:
   python expense_tracker.py
