# Expense Tracker (Python)

A command-line expense tracker built using Python to manage and analyze daily expenses efficiently.

## Features
- Add expenses with amount, category, and date
- View all recorded expenses in a structured format
- Calculate total spending
- Category-wise spending summary
- Delete existing expenses

## Technologies Used
- Python
- Lists and Dictionaries
- Functions and Loops

## How to Run
1. Clone the repository:
   git clone https://github.com/manthanarora03-star/expense-tracker-python.git

2. Navigate to the folder:
   cd expense-tracker-python

3. Run the program:
   python expense_tracker.py

## Project Objective
This project was built to strengthen core Python concepts and apply them to a real-world use case of personal finance management.
